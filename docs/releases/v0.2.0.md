# QuotaBuddy v0.2.0

## Destaques

- Refresh silencioso no Windows: processos auxiliares são criados sem janela de console.
- Histórico local de 7 dias, 30 dias ou todo o período, com gráfico e divisão por modelo.
- Tokens de entrada, cached input, saída e reasoning, sem leitura de conteúdo de conversa.
- Equivalente API estimado com preços versionados, cobertura explícita e modelos desconhecidos sem preço inventado.
- Perfil de atividade agregada da conta Codex: tokens acumulados, pico, streaks, turno mais longo e buckets diários.
- Explicação de quota compartilhada e contexto Hermes sem atribuição artificial por cliente.
- Descoberta segura de Codex, Claude Code e Cursor nas configurações.
- Cache por fingerprint para alternar períodos sem reler arquivos inalterados.

## Privacidade

O frontend recebe apenas contadores, datas, IDs de modelo normalizados, percentuais e estados. Email, account/workspace ID, sessões, caminhos, prompts, respostas, tools e credenciais não atravessam a boundary nativa.

## Distribuição

- Windows x64, instalador NSIS por usuário.
- Sem assinatura de código e sem auto-update.
- O SHA-256 conferido acompanha o instalador no asset `.sha256` da release.
