## Limitações intencionais

- Uso remoto não pode ser atribuído individualmente a Hermes, IDE, Web ou outro cliente.
- Claude individual exige futura bridge `statusLine` opt-in; configuração existente não é alterada.
- Cursor individual não possui API pública oficial de uso; nenhum storage privado é raspado.
- Modelos sem preço público exato não recebem alias ou preço inventado.
- O instalador continua sem assinatura de código e sem auto-update.
